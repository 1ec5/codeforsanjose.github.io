Where to Find Data

Local Data Sources

  • City of San Jose’s existing data catalog (currently being revamped, to be relaunched soon)
  • Code for San Jose’s San Jose Data Inventory (this is a work-in-progress, please add to it!)
  • VTA Open Data Portal
Looking for local government data but cant find it? Post a data request as a Github issue and Code for San Joses municipal liaison, Michelle Thong, will help you track it down, if it exists.

California Data Sources

  • State of California open data portal
  • California Department of Public Health open data portal
  • California Common Sense open data portal
  • California Center for Jobs & the Economy data tool

Federal Data Sources

  • U.S. Government open data portal
  • U.S. Census Data on American Factfinder or via the Census API
  • Third-party sources of U.S Census Data such as Census Reporter and Social Explorer
